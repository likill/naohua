# Frontend Design Skills

This directory contains frontend design and styling skills for the project.

## Available Skills

| Skill | Description |
|-------|-------------|
| `css-styling` | CSS layout, animations, Flexbox/Grid, transitions, glass morphism |
| `ui-design` | UI component patterns, cards, forms, modals, buttons, layout |
| `color-theme` | Color schemes, theming, gradients, semantic colors |
| `responsive-design` | Mobile-first, breakpoints, adaptive layouts |

## Usage

When working on frontend design tasks, invoke the relevant skill:

```
/skill css-styling
/skill ui-design
/skill color-theme
/skill responsive-design
```
